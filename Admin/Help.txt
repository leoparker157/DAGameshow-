1. First, you must have a list of questions, go to Question button to create a file that contains a list of questions located in project folder.

2. Create a schedule, click the Schedule button to create it, txt file contains the schedule is located in the project folder, caution : don't create a schedule that has been expired and don't spam the button , it will create a multiple file distinguish by number .

3. when you already have 2 conditions above , before start the sever click the LoadQuest to upload Schedule to the server.

4. Click the ServerStart button , the game will wait for clients for a time before starting .

5. Left gridView show who has joined or left server , right gridView show comment of clients and Admin. 

6. Quit button may causes serval bug like process not fully closed in task manager .
